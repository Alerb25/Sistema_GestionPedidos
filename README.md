--Resumen del proyecto--
El proyecto se basa en diseñar y crear un programa para la gestion de pedidos de una empresa.
Se deben gestionar productos que se dividen en varios tipos, se aplica herencia, ecxiste un registro de clientes, gestión de pedidos.
También se calcula el importe total (Teniendo en cuenta IVA, descuentos, coste de envios).
Y finalmente se mostrará por pantalla un resumen del pedido.
